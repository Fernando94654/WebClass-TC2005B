# WebClass - School Practice

Simple project to practice web development in class.

## Objective

This project is a school practice exercise to learn the basic structure of a web app:

- Simple frontend (HTML, CSS, JS)
- Backend with Node.js + Express
- Basic REST routes

## Structure

- `fronted/`: frontend pages and assets
- `backend/`: REST API

## Run Backend

From `backend/`:

```bash
npm install
npm start
```

For development:

```bash
npm run dev
```

## Note

This is a practice project, intentionally kept basic for school purposes.
